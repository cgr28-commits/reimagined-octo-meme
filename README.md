# My Airport Taxi NI

Next.js site for Northern Ireland airport transfers — WhatsApp booking, quote form, and airport information.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

- Next.js 15
- React 19
- Tailwind CSS 4
